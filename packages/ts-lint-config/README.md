## Installing

```sh
yarn add -D @martin_hotell/ts-lint-config

# also you need to install peer dependencies
yarn add -D tslint
```

## Getting started

Then in your project's tslint.json:

```json
{
  "extends": "@martin_hotell/ts-lint-config"
}
```
